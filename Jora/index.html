ВоргаТех

Компания «Ворга Интех» производит пластины защитные композитные полимерные анатомические (далее пластина защитная СВМПЭ).

Пластина защитная СВМПЭ характеризуется повышенным уровнем прочности, износостойкости, долговечности и надежности, за счет чего нашли широкий спектр применения как защитный элемент (деталь/сборочная единица) средств индивидуальной или неиндивидуальной бронезащиты, предназначенная для поглощения и (или) рассеивания энергии средств поражения от пуль стрелкового оружия, осколков взрывчатых устройств, холодного оружия.

Пластина защитная СВМПЭ обладает:

  • ✓ Антирикошет;
  • ✓ Нулевым водопоглощением;
  • ✓ Минимальным весом – его плотность ниже, чем аналогичный показатель воды;
  • ✓ Повышенной устойчивостью к истиранию, износу, изгибу;
  • ✓ Высоким показателем удельной прочности;
  • ✓ Минимальной газонепроницаемостью;
  • ✓ Уникальной способностью поглощать и равномерно рассеивать ударную нагрузку;
  • ✓ Экологичностью.

Пластина защитная СВМПЭ при индивидуальной бронезащите предназначена для ношения в индивидуальном бронежилете, для чего вставляется в специальный карман бронежилета.

Бронежилет тактический «ВОРГА -1»

Бронежилет состоит из двух частей: передняя и задняя части, которые соединены боковыми поясами. Передняя и задняя части снабжены карманами, в которые помещаются дополнительные пластины СВМПЭ БР-1, БР-2, БР-3 классы защиты. В боковом поясе по два кармана с каждой стороны для мягкой бронезащиты до БР-3 класса защиты. По одному карману для дополнительных твердых бронеэлементов до БР-5 класса. Двухсторонний пояс можно расстегивать и застегивать любой рукой. На боковом двустороннем поясе прорези сделаны под углом для более удобного размещения снаряжения.

На передней и задней стенках жилета расположены механизмы крепления проводов средств коммуникации. На передней панели есть пазы для крепления магазина или подсумки. Жилет со всех сторон покрыт тканью CORDURA. Внутренние стенки жилета с сетчатой подкладкой 3D с ячейками 1 мм выводят излишнее тепло (выполняют охлаждающую функцию). Сетка 3D с функцией позитивного давления увеличивает комфорт носки.

Плечевые отделы жилета выполнены из камуфляжной тесьмы и покрыты съемными рукавами. Рукава также открываются для пропуска коммуникационных проводов.

Бронежилет «ВОРГА 1» может быть оснащен дополнительными демпферами антибактериальными весом не более 500 грамм для снижения запреградного воздействия. Дополнительный демпфер имеет Сертификат БР-1 класса. Благодаря применению в бронежилете только СВМПЭ, изделие абсолютно влагостойкое и может быть использовано как спасательное средство на воде.
